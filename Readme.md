Этот проект собирается с помощью [Gatsby](https://www.gatsbyjs.com/).

### Доступные скрипты

`npm run start`

Команда для запуска в режиме разработки.<br />
Открыть [http://localhost:8000](http://localhost:8000/)

Открыть [http://localhost:8000/\_\_\_graphql](http://localhost:8000/___graphql)

`npm run netlify`

Команда для запуска проекта через netlify.<br />
Открыть [http://localhost:8888](http://localhost:8888/)

`npm run build`

Команда для сборки проекта для размещения на хостинге
